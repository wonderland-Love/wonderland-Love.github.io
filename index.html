学习编程怎么做，自己会更有信心？

我觉得信心不是来自于掌握了多少技能，而是取决于如何解决问题

在去年，我遇到了一个很棘手的问题，它运行的结果和我的想象的完全不一样。我先是尝试自己解决，但没有成功。我想寻求别人的帮助，但他们都很忙，或者不愿意帮我。我感到很失望，但也激发了我的斗志。我决定自己去拆解和跟踪这个问题，查找相关的文章和资料，多试验自己的猜想，不怕失败。在这个过程中，我学到了很多细节方面的知识，也找到了解决问题的方法。这次之后也增强了我的信心。

面对复杂，怎么做更好呢？

我认为解决繁琐复杂的问题，如果已掌握的方法使用起来繁琐，或有很多的代码复用，这时候那应该去探索更优的解决方案。可以先学习需要用到的知识，然后再尝试解决问题。这样可以提高效率和质量，也可以拓展自己的视野和能力。

比如，在我学习 react 的时候，有两种方式做组件，一种是 class 组件，一种是 function 组件。我一开始只用 class 组件，后来听说 function 组件有了 hooks，可以写出更简洁高效的代码。我开始有些抵触，觉得 function 组件很陌生，很难学。但后来我提高了自学能力和解决问题的能力，我愿意去尝试新的东西，因为我觉得只要认真去了解，就没有什么太难的。我开始学习 function 组件，发现它很简单很舒服，实现功能的时候也更快更好。

我发现自己之前不想学习新的知识，是因为对未知的东西有恐惧感，不知道它有多难。我们不应该满足于现状，而应该不断探索新的知识，这样才能在面对问题时有更多的选择和优势。学完新的知识后，会发现它也没那么难。

对于那些刚接触编程不久的人们有什么建议？

  • 学习编程很考验耐心和自己逻辑思维，在遇到问题时要耐心对待，解决难题的时候要有自己的逻辑思维。不要轻易放弃，在生活中的任何事情也是这样，这可能是它在考验你。
  • 自己在使用编程不知道开发什么项目时，看看生活中有什么问题或需求需要解决，这样你可能会很有兴趣去解决它，还会让你对自己很有成就感。
  • 参加到一些项目当中，模仿其他人的优秀代码并改进自己。并与其他人合作开发某个项目，可以学习到新的东西，还可以享受团队合作的乐趣。
